## 1. 代数式的定义

代数式是由数、字母和运算符号组成的有意义的式子。

## 2. 代数式的基本运算

- 加法、减法、乘法、除法、乘方、开方等。

## 3. 恒等变形的概念

- 恒等变形是指在不改变表达式值域的前提下，通过合法的运算步骤将表达式变为等价形式。
- 例如：$a^2 - b^2 = (a-b)(a+b)$
