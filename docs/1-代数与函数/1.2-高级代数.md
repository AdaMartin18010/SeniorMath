# 1.2 高级代数 (Advanced Algebra)

## 模块概述

高级代数是基础代数的深化和发展，涵盖了从二次函数到复合函数的完整知识体系。本模块重点分析高级代数概念之间的关联性，构建完整的函数知识网络，为微积分和高等数学奠定基础。

## 知识结构

### 1. 二次函数与多项式

#### 1.1 二次函数

- **标准形式**：f(x) = ax² + bx + c (a≠0)
- **顶点形式**：f(x) = a(x-h)² + k
- **性质分析**：
  - 对称轴：x = -b/(2a)
  - 顶点坐标：(-b/(2a), f(-b/(2a)))
  - 开口方向：a > 0 向上，a < 0 向下
  - 单调性：在对称轴左侧递减，右侧递增

#### 1.2 多项式函数

- **一般形式**：f(x) = aₙxⁿ + aₙ₋₁xⁿ⁻¹ + ... + a₁x + a₀
- **多项式性质**：
  - 连续性：多项式函数在实数域上连续
  - 可导性：多项式函数在实数域上可导
  - 根的性质：n次多项式最多有n个实根
- **多项式运算**：加法、减法、乘法、除法

#### 1.3 因式定理与余数定理

- **因式定理**：如果f(a) = 0，则(x-a)是f(x)的因式
- **余数定理**：f(x)除以(x-a)的余数是f(a)
- **综合除法**：多项式除法的简化方法

### 2. 有理函数与渐近线

#### 2.1 有理函数定义

- **一般形式**：f(x) = P(x)/Q(x)，其中P(x)、Q(x)是多项式
- **定义域**：Q(x) ≠ 0 的所有实数
- **性质**：有理函数在其定义域内连续

#### 2.2 渐近线分析

- **垂直渐近线**：x = a，其中Q(a) = 0
- **水平渐近线**：
  - 当deg(P) < deg(Q)时，y = 0
  - 当deg(P) = deg(Q)时，y = 前项系数比
  - 当deg(P) > deg(Q)时，无水平渐近线
- **斜渐近线**：当deg(P) = deg(Q) + 1时存在

#### 2.3 有理函数图像

- **图像特征**：分段连续，可能有间断点
- **单调性分析**：通过导数分析单调性
- **极值分析**：通过导数求极值点

### 3. 指数与对数函数

#### 3.1 指数函数

- **定义**：f(x) = aˣ (a > 0, a ≠ 1)
- **性质**：
  - 定义域：R
  - 值域：(0, +∞)
  - 单调性：a > 1 递增，0 < a < 1 递减
  - 图像：过点(0,1)，无零点
- **指数运算**：
  - aˣ⁺ʸ = aˣ·aʸ
  - aˣ⁻ʸ = aˣ/aʸ
  - (aˣ)ʸ = aˣʸ

#### 3.2 对数函数

- **定义**：f(x) = logₐx (a > 0, a ≠ 1)
- **性质**：
  - 定义域：(0, +∞)
  - 值域：R
  - 单调性：a > 1 递增，0 < a < 1 递减
  - 图像：过点(1,0)，无零点
- **对数运算**：
  - logₐ(xy) = logₐx + logₐy
  - logₐ(x/y) = logₐx - logₐy
  - logₐ(xⁿ) = n·logₐx

#### 3.3 指数对数方程

- **指数方程**：aˣ = b 的解为 x = logₐb
- **对数方程**：logₐx = b 的解为 x = aᵇ
- **复合方程**：结合指数和对数的复杂方程

### 4. 复合函数与变换

#### 4.1 复合函数

- **定义**：f∘g(x) = f(g(x))
- **定义域**：g(x)的定义域与f(x)的定义域的交集
- **性质**：复合函数不一定满足交换律
- **求法**：从内到外逐步计算

#### 4.2 函数变换

- **平移变换**：
  - 水平平移：f(x±a)
  - 垂直平移：f(x)±a
- **伸缩变换**：
  - 水平伸缩：f(ax)
  - 垂直伸缩：af(x)
- **对称变换**：
  - 关于y轴：f(-x)
  - 关于x轴：-f(x)
  - 关于原点：-f(-x)

#### 4.3 反函数

- **定义**：如果f∘f⁻¹(x) = x，则f⁻¹是f的反函数
- **性质**：f和f⁻¹的图像关于y=x对称
- **求法**：交换x和y，解出y
- **存在条件**：f必须是双射函数

## 典型例题与详细解答 | Typical Examples and Detailed Solutions

### 例题1：二次函数与多项式 | Example 1: Quadratic Function and Polynomial

**题目 | Problem**：已知f(x) = x² - 4x + 3，求f(f(2))。
**解答 | Solution**：
f(2) = 2² - 4×2 + 3 = 4 - 8 + 3 = -1
f(f(2)) = f(-1) = (-1)² - 4×(-1) + 3 = 1 + 4 + 3 = 8

### 例题2：有理函数与渐近线 | Example 2: Rational Function and Asymptote

**题目 | Problem**：求函数y = 1/(x-2)的定义域和渐近线。
**解答 | Solution**：
定义域：x ≠ 2。
垂直渐近线：x = 2。
水平渐近线：y = 0。
Domain: x ≠ 2. Vertical asymptote: x = 2. Horizontal asymptote: y = 0.

### 例题3：指数与对数函数 | Example 3: Exponential and Logarithmic Functions

**题目 | Problem**：解关于x的方程 log₂(x - 1) + log₂(3 - x) = 1。
**解答 | Solution**：
令y = x - 1，则3 - x = 2 - y。
log₂(y) + log₂(2 - y) = 1
log₂[y(2 - y)] = 1
y(2 - y) = 2¹ = 2
2y - y² = 2
y² - 2y + 2 = 0
判别式Δ = 4 - 8 < 0，无实数解。
回到原式：x - 1 > 0 且 3 - x > 0 → 1 < x < 3
直接代入x = 2，log₂(1) + log₂(1) = 0 + 0 = 0 ≠ 1
无实数解。
Let y = x - 1, then 3 - x = 2 - y. The equation has no real solution.

### 例题4：复合函数与变换 | Example 4: Composite Function and Transformation

**题目 | Problem**：已知g(x) = 2^x + 2^{-x}，求g(3) + g(-3)。
**解答 | Solution**：
g(3) = 2³ + 2^{-3} = 8 + 1/8 = 8.125
g(-3) = 2^{-3} + 2^{3} = 1/8 + 8 = 8.125
g(3) + g(-3) = 8.125 + 8.125 = 16.25

### 例题5：代数创新与综合 | Example 5: Algebraic Innovation and Synthesis

**题目 | Problem**：已知a + b + c = 0，求证a³ + b³ + c³ = 3abc。
**解答 | Solution**：
由(a + b + c)³ = a³ + b³ + c³ + 3(a + b)(b + c)(c + a)
但更常用推导：
a³ + b³ + c³ - 3abc = (a + b + c)(a² + b² + c² - ab - bc - ca)
因a + b + c = 0，故a³ + b³ + c³ = 3abc。
By the identity, since a + b + c = 0, we have a³ + b³ + c³ = 3abc.

### 例题6：对数与指数综合 | Example 6: Logarithmic and Exponential Synthesis

**题目 | Problem**：解关于x的方程 2^x + 2^{-x} = 5。
**解答 | Solution**：
设y=2^x，则2^{-x}=1/y，y+1/y=5，y²-5y+1=0，y=[5±√21]/2，x=log₂([5±√21]/2)。
Let y=2^x, then 2^{-x}=1/y, y+1/y=5, y²-5y+1=0, y=[5±√21]/2, x=log₂([5±√21]/2).

### 例题7：高次不等式 | Example 7: Higher Order Inequality

**题目 | Problem**：解关于x的不等式 x⁴-5x²+4<0。
**解答 | Solution**：
令y=x²，y²-5y+4<0，解得1<y<4，x²∈(1,4)，x∈(-2,-1)∪(1,2)。
Let y=x², then y²-5y+4<0, 1<y<4, so x²∈(1,4), x∈(-2,-1)∪(1,2).

### 例题8：综合函数性质 | Example 8: Comprehensive Function Properties

**题目 | Problem**：已知f(x)=x²-4x+3，g(x)=2^x，求满足f(x)=g(x)的x的取值范围。
**解答 | Solution**：
令x²-4x+3=2^x，作图或数值法近似求解，x约在(0,1)和(3,4)之间有解。
Let x²-4x+3=2^x, solve graphically or numerically, solutions are approximately in (0,1) and (3,4).
