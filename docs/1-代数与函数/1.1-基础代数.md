# 1.1 基础代数 (Basic Algebra)

## 模块概述

基础代数是高中数学的起点，为整个数学体系奠定基础。本模块系统介绍实数系统、代数运算、方程不等式和函数基础概念，重点分析这些概念之间的关联性和结构性。

## 知识结构

### 1. 实数系统与数轴

#### 1.1 实数分类

```text
实数 (R)
├── 有理数 (Q)
│   ├── 整数 (Z)
│   │   ├── 正整数 (N)
│   │   ├── 零 (0)
│   │   └── 负整数
│   └── 分数
└── 无理数
    ├── 代数无理数
    └── 超越无理数
```

#### 1.2 数轴与坐标系

- **数轴性质**：稠密性、完备性、有序性
- **坐标表示**：点的坐标表示方法
- **距离概念**：数轴上两点间距离的计算

#### 1.3 绝对值与不等式

- **绝对值定义**：|a| = a (a≥0) 或 |a| = -a (a<0)
- **绝对值性质**：|ab| = |a|·|b|, |a+b| ≤ |a| + |b|
- **绝对值不等式**：|x-a| < b 的几何意义

### 2. 代数运算与因式分解

#### 2.1 多项式运算

- **多项式定义**：形如 aₙxⁿ + aₙ₋₁xⁿ⁻¹ + ... + a₁x + a₀ 的表达式
- **多项式分类**：按次数分类（常数、一次、二次等）
- **多项式运算**：加法、减法、乘法运算

#### 2.2 因式分解方法

- **提取公因式**：ax + ay = a(x + y)
- **平方差公式**：a² - b² = (a+b)(a-b)
- **完全平方公式**：a² ± 2ab + b² = (a±b)²
- **十字相乘法**：二次三项式的因式分解
- **分组分解法**：四项及以上的因式分解

#### 2.3 分式运算

- **分式定义**：两个多项式的比值
- **分式化简**：约分、通分、化简
- **分式运算**：加减乘除运算

### 3. 方程与不等式

#### 3.1 线性方程

- **一元一次方程**：ax + b = 0 (a≠0)
- **解的性质**：唯一解、无解、无穷多解
- **应用问题**：实际问题建模与求解

#### 3.2 二次方程

- **标准形式**：ax² + bx + c = 0 (a≠0)
- **求根公式**：x = (-b ± √(b²-4ac))/(2a)
- **判别式**：Δ = b² - 4ac
- **根与系数关系**：韦达定理

#### 3.3 不等式

- **一元一次不等式**：ax + b > 0 等
- **一元二次不等式**：ax² + bx + c > 0 等
- **不等式解法**：数轴法、配方法、判别式法

### 4. 函数基础概念

#### 4.1 函数定义

- **函数概念**：两个集合之间的对应关系
- **定义域与值域**：函数的输入和输出范围
- **函数表示**：解析式、表格、图像表示

#### 4.2 函数性质

- **单调性**：增函数、减函数
- **奇偶性**：奇函数、偶函数
- **周期性**：周期函数的概念

#### 4.3 基本初等函数

- **常数函数**：f(x) = c
- **一次函数**：f(x) = ax + b
- **二次函数**：f(x) = ax² + bx + c
- **反比例函数**：f(x) = k/x

## 典型例题与解题思路

### 例题1：因式分解

**题目**：分解因式 x³ - 8

**解题思路**：

1. 识别立方差公式：a³ - b³ = (a-b)(a² + ab + b²)
2. 应用公式：x³ - 8 = x³ - 2³ = (x-2)(x² + 2x + 4)

### 例题2：二次方程

**题目**：解方程 x² - 5x + 6 = 0

**解题思路**：

1. 判别式：Δ = 25 - 24 = 1 > 0
2. 求根公式：x = (5 ± √1)/2 = 3 或 2
3. 验证：3² - 5×3 + 6 = 0, 2² - 5×2 + 6 = 0

### 例题3：函数性质

**题目**：判断函数 f(x) = x² + 2x + 1 的单调性

**解题思路**：

1. 配方法：f(x) = (x+1)²
2. 分析：当 x < -1 时递减，当 x > -1 时递增
3. 结论：在 (-∞, -1] 递减，在 [-1, +∞) 递增
