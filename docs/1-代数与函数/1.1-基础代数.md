# 1.1 基础代数 (Basic Algebra)

## 模块概述

基础代数是高中数学的起点，为整个数学体系奠定基础。本模块系统介绍实数系统、代数运算、方程不等式和函数基础概念，重点分析这些概念之间的关联性和结构性。

## 知识结构

### 1. 实数系统与数轴

#### 1.1 实数分类

```text
实数 (R)
├── 有理数 (Q)
│   ├── 整数 (Z)
│   │   ├── 正整数 (N)
│   │   ├── 零 (0)
│   │   └── 负整数
│   └── 分数
└── 无理数
    ├── 代数无理数
    └── 超越无理数
```

#### 1.2 数轴与坐标系

- **数轴性质**：稠密性、完备性、有序性
- **坐标表示**：点的坐标表示方法
- **距离概念**：数轴上两点间距离的计算

#### 1.3 绝对值与不等式

- **绝对值定义**：|a| = a (a≥0) 或 |a| = -a (a<0)
- **绝对值性质**：|ab| = |a|·|b|, |a+b| ≤ |a| + |b|
- **绝对值不等式**：|x-a| < b 的几何意义

### 2. 代数运算与因式分解

#### 2.1 多项式运算

- **多项式定义**：形如 aₙxⁿ + aₙ₋₁xⁿ⁻¹ + ... + a₁x + a₀ 的表达式
- **多项式分类**：按次数分类（常数、一次、二次等）
- **多项式运算**：加法、减法、乘法运算

#### 2.2 因式分解方法

- **提取公因式**：ax + ay = a(x + y)
- **平方差公式**：a² - b² = (a+b)(a-b)
- **完全平方公式**：a² ± 2ab + b² = (a±b)²
- **十字相乘法**：二次三项式的因式分解
- **分组分解法**：四项及以上的因式分解

#### 2.3 分式运算

- **分式定义**：两个多项式的比值
- **分式化简**：约分、通分、化简
- **分式运算**：加减乘除运算

### 3. 方程与不等式

#### 3.1 线性方程

- **一元一次方程**：ax + b = 0 (a≠0)
- **解的性质**：唯一解、无解、无穷多解
- **应用问题**：实际问题建模与求解

#### 3.2 二次方程

- **标准形式**：ax² + bx + c = 0 (a≠0)
- **求根公式**：x = (-b ± √(b²-4ac))/(2a)
- **判别式**：Δ = b² - 4ac
- **根与系数关系**：韦达定理

#### 3.3 不等式

- **一元一次不等式**：ax + b > 0 等
- **一元二次不等式**：ax² + bx + c > 0 等
- **不等式解法**：数轴法、配方法、判别式法

### 4. 函数基础概念

#### 4.1 函数定义

- **函数概念**：两个集合之间的对应关系
- **定义域与值域**：函数的输入和输出范围
- **函数表示**：解析式、表格、图像表示

#### 4.2 函数性质

- **单调性**：增函数、减函数
- **奇偶性**：奇函数、偶函数
- **周期性**：周期函数的概念

#### 4.3 基本初等函数

- **常数函数**：f(x) = c
- **一次函数**：f(x) = ax + b
- **二次函数**：f(x) = ax² + bx + c
- **反比例函数**：f(x) = k/x

## 典型例题与详细解答

### 例题1：实数与数轴

**题目**：在数轴上表示-3, 0, 2.5的位置。
**解答**：

- 画一条水平直线，标记原点0。
- 向左标记-3，向右标记2.5。
- 每个点的位置与原点的距离分别为3和2.5。

### 例题2：代数式运算

**题目**：化简：(2x + 3) - (x - 5)。
**解答**：
(2x + 3) - (x - 5) = 2x + 3 - x + 5 = x + 8。

### 例题3：因式分解

**题目**：分解因式：x² - 5x + 6。
**解答**：
x² - 5x + 6 = (x - 2)(x - 3)。

### 例题4：一元一次方程

**题目**：解方程：2x - 3 = 7。
**解答**：
2x - 3 = 7
→ 2x = 10
→ x = 5

### 例题5：一元二次不等式

**题目**：解不等式：x² - 4x < 0。
**解答**：
x² - 4x < 0
→ x(x - 4) < 0
→ 解得 0 < x < 4。

### 例题6：函数基础

**题目**：已知f(x) = 2x + 1，求f(3)。
**解答**：
f(3) = 2×3 + 1 = 7。

### 例题7：函数类型判断

**题目**：判断y = x²是否为一次函数。
**解答**：
y = x²是二次函数，不是一元一次函数。

## 创新与挑战性例题

### 例题8：绝对值不等式

**题目**：解关于x的不等式 |2x-3|<5。
**解答**：
-5<2x-3<5 → -2<x<4。

### 例题9：分式方程

**题目**：解方程 1/(x-1) + 1/(x+1) = 1。
**解答**：
通分得：(x+1)+(x-1)=(x-1)(x+1)
2x=x²-1
x²-2x-1=0
x=1±√2
（x≠±1）

### 例题10：函数综合应用

**题目**：已知f(x)=x²-2x+1，求使f(x)<4的x的取值范围。
**解答**：
x²-2x+1<4
x²-2x-3<0
(x-3)(x+1)<0
-1<x<3
