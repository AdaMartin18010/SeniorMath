# 1.3 三角函数 (Trigonometric Functions)

## 模块概述

三角函数是连接代数与几何的重要桥梁，在数学、物理、工程等领域有广泛应用。本节系统介绍三角函数的定义、性质、图像、恒等式和应用，强调知识递进、结构清晰、实际应用与创新能力培养。

- 拓展：复数三角函数、傅里叶分析、国际竞赛专题
