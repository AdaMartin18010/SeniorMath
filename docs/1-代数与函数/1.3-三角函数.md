# 1.3 三角函数 (Trigonometric Functions)

## 模块概述

三角函数是连接代数与几何的重要桥梁，在数学、物理、工程等领域有广泛应用。本模块系统介绍三角函数的定义、性质、图像、恒等式和应用，重点分析三角函数与其他数学概念之间的关联性。

## 知识结构

### 1. 三角函数定义与性质

#### 1.1 基本三角函数

- **正弦函数**：sin θ = y/r
- **余弦函数**：cos θ = x/r
- **正切函数**：tan θ = y/x = sin θ/cos θ
- **余切函数**：cot θ = x/y = cos θ/sin θ
- **正割函数**：sec θ = r/x = 1/cos θ
- **余割函数**：csc θ = r/y = 1/sin θ

#### 1.2 三角函数定义域与值域

- **sin θ, cos θ**：
  - 定义域：R
  - 值域：[-1, 1]
- **tan θ, sec θ**：
  - 定义域：θ ≠ π/2 + kπ (k∈Z)
  - 值域：R (tan θ), (-∞, -1]∪[1, +∞) (sec θ)
- **cot θ, csc θ**：
  - 定义域：θ ≠ kπ (k∈Z)
  - 值域：R (cot θ), (-∞, -1]∪[1, +∞) (csc θ)

#### 1.3 三角函数基本性质

- **周期性**：
  - sin θ, cos θ：周期为2π
  - tan θ, cot θ：周期为π
- **奇偶性**：
  - 奇函数：sin θ, tan θ, cot θ, csc θ
  - 偶函数：cos θ, sec θ
- **单调性**：在特定区间内的单调性

### 2. 三角函数图像与周期

#### 2.1 正弦函数图像

- **基本图像**：y = sin x
- **图像特征**：
  - 过原点(0,0)
  - 最大值1，最小值-1
  - 在[0,2π]上的变化规律
- **变换图像**：y = A sin(Bx + C) + D

#### 2.2 余弦函数图像

- **基本图像**：y = cos x
- **图像特征**：
  - 过点(0,1)
  - 最大值1，最小值-1
  - 与正弦函数的关系：cos x = sin(x + π/2)
- **变换图像**：y = A cos(Bx + C) + D

#### 2.3 正切函数图像

- **基本图像**：y = tan x
- **图像特征**：
  - 过原点(0,0)
  - 垂直渐近线：x = π/2 + kπ
  - 值域：R
- **变换图像**：y = A tan(Bx + C) + D

#### 2.4 函数变换

- **振幅变换**：y = A sin x (A为振幅)
- **周期变换**：y = sin(Bx) (周期为2π/|B|)
- **相位变换**：y = sin(x + C) (向左平移C个单位)
- **垂直平移**：y = sin x + D (向上平移D个单位)

### 3. 三角函数恒等式

#### 3.1 基本恒等式

- **毕达哥拉斯恒等式**：
  - sin²θ + cos²θ = 1
  - 1 + tan²θ = sec²θ
  - 1 + cot²θ = csc²θ
- **倒数关系**：
  - sin θ = 1/csc θ
  - cos θ = 1/sec θ
  - tan θ = 1/cot θ

#### 3.2 和差公式

- **正弦和差公式**：
  - sin(A+B) = sin A cos B + cos A sin B
  - sin(A-B) = sin A cos B - cos A sin B
- **余弦和差公式**：
  - cos(A+B) = cos A cos B - sin A sin B
  - cos(A-B) = cos A cos B + sin A sin B
- **正切和差公式**：
  - tan(A+B) = (tan A + tan B)/(1 - tan A tan B)
  - tan(A-B) = (tan A - tan B)/(1 + tan A tan B)

#### 3.3 倍角公式

- **正弦倍角公式**：
  - sin(2A) = 2 sin A cos A
- **余弦倍角公式**：
  - cos(2A) = cos²A - sin²A = 2cos²A - 1 = 1 - 2sin²A
- **正切倍角公式**：
  - tan(2A) = 2tan A/(1 - tan²A)

#### 3.4 半角公式

- **正弦半角公式**：
  - sin(A/2) = ±√[(1 - cos A)/2]
- **余弦半角公式**：
  - cos(A/2) = ±√[(1 + cos A)/2]
- **正切半角公式**：
  - tan(A/2) = ±√[(1 - cos A)/(1 + cos A)]

### 4. 三角函数应用

#### 4.1 解三角形

- **正弦定理**：a/sin A = b/sin B = c/sin C = 2R
- **余弦定理**：c² = a² + b² - 2ab cos C
- **面积公式**：S = ½ab sin C
- **解三角形方法**：
  - 已知三边：用余弦定理求角
  - 已知两边一角：用正弦定理求其他边角
  - 已知两角一边：用正弦定理求其他边

#### 4.2 实际问题应用

- **测量问题**：高度、距离的测量
- **航海问题**：方位角、距离的计算
- **建筑问题**：角度、高度的计算
- **物理问题**：振动、波动、力的分解

#### 4.3 三角方程

- **基本三角方程**：
  - sin x = a (|a| ≤ 1)
  - cos x = a (|a| ≤ 1)
  - tan x = a
- **复合三角方程**：
  - A sin(Bx + C) = D
  - A cos(Bx + C) = D
- **三角不等式**：结合代数不等式求解

## 典型例题与详细解答 | Typical Examples and Detailed Solutions

### 例题1：三角函数定义与性质 | Example 1: Definition and Properties of Trigonometric Functions

**题目 | Problem**：已知sinθ = 1/2，0° ≤ θ < 360°，求θ的所有值。
**解答 | Solution**：
sinθ = 1/2 → θ = 30° 或 150°。
θ = 30° or 150°.

### 例题2：三角函数图像与周期 | Example 2: Graph and Period of Trigonometric Functions

**题目 | Problem**：写出y = sin(x)的周期与对称轴。
**解答 | Solution**：
周期为2π，对称轴为x = kπ (k为整数)。
The period is 2π, and the axes of symmetry are x = kπ (k is an integer).

### 例题3：三角恒等式 | Example 3: Trigonometric Identity

**题目 | Problem**：证明：sin²x + cos²x = 1。
**解答 | Solution**：
由单位圆定义，任意x有sin²x + cos²x = 1。
By the definition of the unit circle, for any x, sin²x + cos²x = 1.

### 例题4：三角函数应用 | Example 4: Application of Trigonometric Functions

**题目 | Problem**：已知一个直角三角形的一个锐角为30°，斜边长为10，求对边和邻边的长度。
**解答 | Solution**：
对边 = 10 × sin30° = 10 × 1/2 = 5
邻边 = 10 × cos30° = 10 × (√3/2) = 5√3
Opposite side = 10 × sin30° = 5; Adjacent side = 10 × cos30° = 5√3.

### 例题5：三角函数方程 | Example 5: Trigonometric Equation

**题目 | Problem**：解方程2sinx - 1 = 0，x∈[0,2π)。
**解答 | Solution**：
2sinx - 1 = 0 → sinx = 1/2
x = π/6 或 5π/6
x = π/6 or 5π/6.

## 创新与挑战性例题 | Innovative and Challenging Examples

### 例题6：三角恒等变换 | Example 6: Trigonometric Identity Transformation

**题目 | Problem**：化简表达式 sinx·cosx。
**解答 | Solution**：
sinx·cosx=1/2·sin2x。
sinx·cosx = 1/2·sin2x.

### 例题7：三角函数方程综合 | Example 7: Comprehensive Trigonometric Equation

**题目 | Problem**：解方程 sin2x=√3cosx，x∈[0,2π)。
**解答 | Solution**：
sin2x=2sinxcosx=√3cosx
2sinxcosx-√3cosx=0
cosx(2sinx-√3)=0
cosx=0 或 sinx=√3/2
cosx=0→x=π/2,3π/2
sinx=√3/2→x=π/3,2π/3
cosx=0 or sinx=√3/2; x=π/2, 3π/2, π/3, 2π/3.

### 例题8：三角函数应用创新 | Example 8: Innovative Application of Trigonometric Functions

**题目 | Problem**：已知tanα=2，求sinα+cosα的最大值。
**解答 | Solution**：
tanα=2，设sinα=2cosα，sinα+cosα=2cosα+cosα=3cosα
|sinα|≤1，cosα=sinα/2，sinα=2cosα，代入sin²α+cos²α=1
(2cosα)²+cos²α=1→4cos²α+cos²α=1→5cos²α=1→cosα=±1/√5
sinα=±2/√5
sinα+cosα最大值=2/√5+1/√5=3/√5
Let tanα=2, set sinα=2cosα, sinα+cosα=3cosα. Substitute into sin²α+cos²α=1, get cosα=±1/√5, sinα=±2/√5, so the maximum value is 3/√5.
