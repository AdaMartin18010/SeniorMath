## 1. 子集的定义

若集合 $A$ 的任一元素都是集合 $B$ 的元素，则称 $A$ 是 $B$ 的子集，记作 $A \subseteq B$。

## 2. 集合的包含关系

- 真子集：$A \subset B$ 且 $A \neq B$
- 空集是任意集合的子集
- 全集包含所有集合

## 3. 集合的基本运算

- 并集：$A \cup B = \{x \mid x \in A \text{ 或 } x \in B\}$
- 交集：$A \cap B = \{x \mid x \in A \text{ 且 } x \in B\}$
- 差集：$A - B = \{x \mid x \in A \text{ 且 } x \notin B\}$
- 补集：$A^c = U - A$
