## 3. 代码实现 | Code Implementation

### 3.1 Rust 示例 | Rust Example

```rust
fn f(x: i32) -> i32 { 2 * x }
let y = f(3); // y = 6
```

### 3.2 Lean 示例 | Lean Example

```lean
example : function.injective (λ x : ℕ, 2 * x) :=
begin
  intros a b h,
  exact nat.eq_of_mul_eq_mul_left zero_lt_two h,
end
```
