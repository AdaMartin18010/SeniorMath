# 2.1 平面几何 (Plane Geometry)

## 模块概述

平面几何是几何学的基础，研究平面内几何图形及其性质。本模块系统介绍基本几何概念、三角形、四边形、圆以及相似与全等理论，重点分析平面几何概念之间的关联性和结构性。

## 知识结构

### 1. 基本几何概念与公理

#### 1.1 基本概念

- **点**：几何空间中的基本元素，无大小，只有位置
- **线**：由无数点组成的几何对象
  - **直线**：无限延伸的一维几何对象
  - **射线**：从一点出发无限延伸的半直线
  - **线段**：直线上两点之间的有限部分
- **面**：二维几何对象
  - **平面**：无限延伸的二维几何对象
  - **多边形**：由有限条线段围成的平面图形

#### 1.2 基本公理

- **欧几里得公理**：
  - 两点确定一条直线
  - 直线可以无限延伸
  - 以任意点为圆心，任意距离为半径可以画圆
  - 所有直角都相等
  - 平行公理：过直线外一点有且仅有一条平行线

#### 1.3 基本性质

- **距离**：两点间的最短距离
- **角度**：两条射线之间的夹角
- **平行**：两条直线永不相交
- **垂直**：两条直线相交成直角

### 2. 三角形

#### 2.1 三角形分类

- **按边长分类**：
  - 等边三角形：三边相等
  - 等腰三角形：两边相等
  - 不等边三角形：三边都不相等
- **按角度分类**：
  - 锐角三角形：三个角都是锐角
  - 直角三角形：有一个直角
  - 钝角三角形：有一个钝角

#### 2.2 三角形性质

- **内角和**：三角形内角和为180°
- **外角和**：三角形外角和为360°
- **边角关系**：
  - 大边对大角
  - 等边对等角
- **特殊线段**：
  - **中线**：连接顶点和对边中点的线段
  - **高线**：从顶点到对边的垂线
  - **角平分线**：平分内角的射线
  - **垂直平分线**：垂直平分边的直线

#### 2.3 三角形面积

- **基本公式**：S = ½bh（底×高÷2）
- **海伦公式**：S = √[s(s-a)(s-b)(s-c)]，其中s = (a+b+c)/2
- **三角函数公式**：S = ½ab sin C

#### 2.4 特殊三角形

- **等边三角形**：
  - 三边相等，三角相等（60°）
  - 高线、中线、角平分线重合
- **等腰三角形**：
  - 两腰相等，两底角相等
  - 顶角平分线、高线、中线重合
- **直角三角形**：
  - 勾股定理：a² + b² = c²
  - 特殊角：30°-60°-90°，45°-45°-90°

### 3. 四边形

#### 3.1 四边形分类

- **平行四边形**：对边平行
- **矩形**：四个角都是直角
- **菱形**：四边相等
- **正方形**：四边相等且四个角都是直角
- **梯形**：一组对边平行
- **一般四边形**：无特殊性质

#### 3.2 平行四边形性质

- **对边平行且相等**
- **对角相等**
- **对角线互相平分**
- **面积公式**：S = bh（底×高）

#### 3.3 特殊四边形

- **矩形**：
  - 四个角都是90°
  - 对角线相等
  - 面积：S = ab（长×宽）
- **菱形**：
  - 四边相等
  - 对角线互相垂直
  - 面积：S = ½d₁d₂（对角线乘积的一半）
- **正方形**：
  - 四边相等且四个角都是90°
  - 对角线相等且互相垂直
  - 面积：S = a²（边长的平方）

### 4. 圆

#### 4.1 圆的基本概念

- **圆心**：圆的中心点
- **半径**：圆心到圆上任意点的距离
- **直径**：通过圆心的弦
- **弦**：圆上任意两点间的线段
- **弧**：圆上两点间的部分
- **圆周角**：顶点在圆上，两边经过圆上两点的角

#### 4.2 圆的性质

- **圆心角定理**：圆心角是圆周角的两倍
- **圆周角定理**：同弧圆周角相等
- **切线性质**：切线与半径垂直
- **弦长公式**：l = 2r sin(θ/2)
- **弧长公式**：L = rθ（θ为弧度）

#### 4.3 圆的计算

- **周长**：C = 2πr
- **面积**：S = πr²
- **扇形面积**：S = ½r²θ（θ为弧度）
- **弓形面积**：扇形面积减去三角形面积

### 5. 相似与全等

#### 5.1 全等三角形

- **全等定义**：两个三角形完全重合
- **全等条件**：
  - SSS：三边对应相等
  - SAS：两边及夹角对应相等
  - ASA：两角及夹边对应相等
  - AAS：两角及一边对应相等
  - HL：直角三角形斜边和直角边对应相等

#### 5.2 相似三角形

- **相似定义**：两个三角形对应角相等，对应边成比例
- **相似条件**：
  - AA：两个角对应相等
  - SAS：两边成比例且夹角相等
  - SSS：三边成比例
- **相似性质**：
  - 对应边成比例
  - 对应高、中线、角平分线成比例
  - 面积比等于相似比的平方

#### 5.3 相似多边形

- **相似定义**：对应角相等，对应边成比例
- **相似性质**：
  - 周长比等于相似比
  - 面积比等于相似比的平方
  - 体积比等于相似比的立方

## 典型例题与详细解答

### 例题1：三角形基本性质

**题目**：已知△ABC中，AB = AC，求证：∠B = ∠C。
**解答**：
等腰三角形的两个底角相等，故∠B = ∠C。

### 例题2：矩形对角线

**题目**：求矩形的对角线长度，已知长为6，宽为8。
**解答**：
对角线长度=√(6²+8²)=√(36+64)=√100=10。

### 例题3：三角形类型判定

**题目**：已知三角形三边长分别为3, 4, 5，判断其类型。
**解答**：
3²+4²=9+16=25=5²，故为直角三角形。

### 例题4：平行四边形面积

**题目**：求平行四边形面积，已知底8，高5。
**解答**：
面积=底×高=8×5=40。

### 例题5：圆的切线方程

**题目**：已知圆心O，半径5，求过点A(3,4)的切线方程。
**解答**：
点A到圆心O(0,0)距离为√(3²+4²)=5，A在圆上。
切线方程：3x+4y=25。

### 例题6：圆上点判定

**题目**：已知圆x² + y² = 25，判断点(3,4)是否在圆上。
**解答**：
3²+4²=9+16=25，点(3,4)在圆上。

## 创新与挑战性例题

### 例题7：圆的切线与半径
**题目**：已知圆x²+y²=25，点P(3,4)在圆上，求过P的切线方程。
**解答**：
切线斜率为-3/4或4/3，切线方程3x+4y=25。

### 例题8：三角形外接圆
**题目**：已知△ABC三边长为5,6,7，求其外接圆半径。
**解答**：
设a=5,b=6,c=7，s=(5+6+7)/2=9，面积S=√[9×4×3×2]=√216=6√6
R=abc/(4S)=5×6×7/(4×6√6)=210/(24√6)=35/(4√6)

### 例题9：几何综合应用
**题目**：已知矩形长为a，宽为b，求对角线与一条边所成角的正弦值。
**解答**：
对角线d=√(a²+b²)，sinθ=b/d=b/√(a²+b²)
