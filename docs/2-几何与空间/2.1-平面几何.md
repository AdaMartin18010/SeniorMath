# 平面几何概念形式模型

## 一、基本几何概念形式化定义

### 1.1 点、线、面的形式化定义

**点的形式化定义**：

```text
点 P ∈ ℝ² 是一个有序实数对 (x, y)，其中 x, y ∈ ℝ
符号表示：P = (x, y)
性质：点没有大小，只有位置
```

**直线的形式化定义**：

```text
直线 l 是满足方程 ax + by + c = 0 的所有点 (x, y) 的集合
其中 a, b, c ∈ ℝ，且 a² + b² ≠ 0
符号表示：l = {(x, y) | ax + by + c = 0}
性质：直线是无限延伸的一维几何对象
```

**平面的形式化定义**：

```text
平面 π 是满足方程 ax + by + cz + d = 0 的所有点 (x, y, z) 的集合
其中 a, b, c, d ∈ ℝ，且 a² + b² + c² ≠ 0
符号表示：π = {(x, y, z) | ax + by + cz + d = 0}
性质：平面是无限延伸的二维几何对象
```

### 1.2 角的形式化定义

**角的形式化定义**：

```text
角 ∠ABC 是由两条射线 BA 和 BC 形成的几何对象
其中 B 是角的顶点，BA 和 BC 是角的两边
符号表示：∠ABC = {射线 BA, 射线 BC}
性质：角的大小用弧度或度数度量
```

**角的度量**：

```text
角 ∠ABC 的大小 θ 满足：
θ = arccos((BA · BC) / (|BA| · |BC|))
其中 BA 和 BC 是向量，· 表示点积，|·| 表示向量的模
```

### 1.3 多边形的形式化定义

**三角形的形式化定义**：

```text
三角形 ΔABC 是由三个不共线的点 A, B, C 及其连线构成的几何对象
符号表示：ΔABC = {点 A, B, C; 线段 AB, BC, CA}
性质：三角形有三个顶点、三条边、三个内角
```

**四边形的形式化定义**：

```text
四边形 ABCD 是由四个点 A, B, C, D 及其连线构成的几何对象
符号表示：ABCD = {点 A, B, C, D; 线段 AB, BC, CD, DA}
性质：四边形有四个顶点、四条边、四个内角
```

## 二、几何变换的形式化定义

### 2.1 平移变换

**平移变换的形式化定义**：

```text
平移变换 T_v 将点 P = (x, y) 映射到点 P' = (x + v_x, y + v_y)
其中 v = (v_x, v_y) 是平移向量
符号表示：T_v(P) = P + v
性质：平移保持距离和角度不变
```

### 2.2 旋转变换

**旋转变换的形式化定义**：

```text
旋转变换 R_θ 将点 P = (x, y) 映射到点 P' = (x', y')
其中：
x' = x cos θ - y sin θ
y' = x sin θ + y cos θ
θ 是旋转角度
符号表示：R_θ(P) = (x cos θ - y sin θ, x sin θ + y cos θ)
性质：旋转保持距离和角度不变
```

### 2.3 缩放变换

**缩放变换的形式化定义**：

```text
缩放变换 S_k 将点 P = (x, y) 映射到点 P' = (kx, ky)
其中 k 是缩放因子
符号表示：S_k(P) = kP
性质：缩放改变距离但保持角度不变
```

## 三、几何定理的形式化证明

### 3.1 勾股定理的形式化证明

**勾股定理**：

```text
在直角三角形中，直角边的平方和等于斜边的平方
即：a² + b² = c²
```

**形式化证明**：

```text
设直角三角形 ΔABC，∠C = 90°
设 BC = a, AC = b, AB = c

根据向量点积：
AB · AB = (AC + CB) · (AC + CB)
        = AC · AC + 2AC · CB + CB · CB
        = |AC|² + 2AC · CB + |CB|²

由于 AC ⊥ CB，所以 AC · CB = 0
因此：|AB|² = |AC|² + |CB|²
即：c² = b² + a²
```

### 3.2 相似三角形的形式化定义

**相似三角形的形式化定义**：

```text
两个三角形 ΔABC 和 ΔDEF 相似，记作 ΔABC ~ ΔDEF
当且仅当：
1. 对应角相等：∠A = ∠D, ∠B = ∠E, ∠C = ∠F
2. 对应边成比例：AB/DE = BC/EF = CA/FD = k
其中 k 是相似比
```

## 四、几何推理系统

### 4.1 基本推理规则

**全等三角形的判定**：

```text
SSS（边边边）：三边对应相等
SAS（边角边）：两边及其夹角对应相等
ASA（角边角）：两角及其夹边对应相等
AAS（角角边）：两角及其中一角的对边对应相等
```

**相似三角形的判定**：

```text
AA（角角）：两角对应相等
SSS（边边边）：三边对应成比例
SAS（边角边）：两边成比例且夹角相等
```

### 4.2 几何证明方法

**直接证明法**：

```text
从已知条件出发，通过逻辑推理直接得出结论
步骤：
1. 列出已知条件
2. 应用几何定理和性质
3. 通过逻辑推理得出结论
```

**反证法**：

```text
假设结论不成立，通过逻辑推理得出矛盾
步骤：
1. 假设结论不成立
2. 从假设出发进行推理
3. 得出与已知条件矛盾的结论
4. 因此原结论成立
```

## 五、几何应用实例

### 5.1 实际测量应用

**高度测量**：

```text
利用相似三角形测量建筑物高度
设建筑物高度为 h，测量距离为 d，测量角度为 θ
则：h = d · tan θ
```

**距离测量**：

```text
利用勾股定理测量不可直接测量的距离
设两点间水平距离为 a，垂直距离为 b
则两点间距离为：c = √(a² + b²)
```
