# 概率与统计 (Probability and Statistics)

## 模块概述

概率与统计是现代数学与数据科学的核心模块，涵盖随机事件、概率、随机变量、分布、统计推断、时间序列、贝叶斯统计、非参数统计、实验设计等内容，强调理论与实际结合、创新应用与国际视野，是连接建模、AI、数据科学等领域的桥梁。

## 知识结构

- 4.1 随机事件与概率（随机试验、样本空间、概率定义与性质）
- 4.2 随机变量与分布（离散与连续、分布律、常见分布、参数估计）
- 4.3 统计推断（参数估计、假设检验、回归分析、方差分析）
- 4.4 时间序列分析
- 4.5 贝叶斯统计
- 4.6 非参数统计
- 4.7 实验设计
- 4.8 统计质量控制
- 4.9 统计学习理论
- 拓展：数据科学、AI、国际竞赛专题

## 典型内容

- 概率的基本概念、性质、计算方法
- 随机变量、分布律、常见分布（如二项、正态、泊松等）
- 参数估计、假设检验、回归分析、方差分析、时间序列、贝叶斯方法
- 典型例题、创新题型、国际竞赛题（IMO、AMC、AIME等）
