# 概率统计概念形式模型

## 一、概率概念的形式化定义

### 1.1 样本空间与事件的形式化定义

**样本空间的形式化定义**：

```
样本空间 Ω 是随机试验所有可能结果的集合
符号表示：Ω = {ω₁, ω₂, ..., ωₙ}
性质：样本空间是完备的，包含所有可能结果
```

**事件的形式化定义**：

```
事件 A 是样本空间 Ω 的子集
符号表示：A ⊆ Ω
性质：事件是样本空间中某些结果的集合
```

**基本事件的形式化定义**：

```
基本事件 {ω} 是样本空间中的单个元素
符号表示：{ω} ∈ Ω
性质：基本事件是不可再分的事件
```

### 1.2 概率测度的形式化定义

**概率测度的形式化定义**：

```
概率测度 P 是从事件集合到实数集的函数
P: 2^Ω → [0, 1]
满足以下公理：
1. 非负性：P(A) ≥ 0，对所有 A ⊆ Ω
2. 规范性：P(Ω) = 1
3. 可加性：对互斥事件 A₁, A₂, ..., Aₙ
   P(∪ᵢAᵢ) = ΣᵢP(Aᵢ)
```

**概率的基本性质**：

```
1. P(∅) = 0
2. P(Aᶜ) = 1 - P(A)
3. 如果 A ⊆ B，则 P(A) ≤ P(B)
4. P(A ∪ B) = P(A) + P(B) - P(A ∩ B)
```

### 1.3 条件概率的形式化定义

**条件概率的形式化定义**：

```
在事件 B 发生的条件下，事件 A 发生的概率
P(A|B) = P(A ∩ B) / P(B)
其中 P(B) > 0
```

**贝叶斯定理的形式化表达**：

```
P(A|B) = P(B|A) · P(A) / P(B)
其中 P(B) = ΣᵢP(B|Aᵢ) · P(Aᵢ)
```

## 二、随机变量的形式化定义

### 2.1 离散随机变量

**离散随机变量的形式化定义**：

```
离散随机变量 X 是从样本空间 Ω 到可数实数集的函数
X: Ω → ℝ
使得对每个实数 x，{ω ∈ Ω | X(ω) = x} 是事件
```

**离散随机变量的概率质量函数**：

```
概率质量函数 p_X(x) = P(X = x)
满足：
1. p_X(x) ≥ 0，对所有 x ∈ ℝ
2. Σᵢp_X(xᵢ) = 1
```

### 2.2 连续随机变量

**连续随机变量的形式化定义**：

```
连续随机变量 X 是从样本空间 Ω 到实数集的函数
X: Ω → ℝ
使得对每个实数 x，{ω ∈ Ω | X(ω) ≤ x} 是事件
```

**连续随机变量的概率密度函数**：

```
概率密度函数 f_X(x) 满足：
P(a ≤ X ≤ b) = ∫ₐᵇ f_X(x) dx
性质：
1. f_X(x) ≥ 0，对所有 x ∈ ℝ
2. ∫₋∞^∞ f_X(x) dx = 1
```

### 2.3 随机变量的数字特征

**期望的形式化定义**：

```
离散随机变量：E[X] = Σᵢxᵢp_X(xᵢ)
连续随机变量：E[X] = ∫₋∞^∞ xf_X(x) dx
```

**方差的形式化定义**：

```
Var(X) = E[(X - E[X])²] = E[X²] - (E[X])²
```

**协方差的形式化定义**：

```
Cov(X, Y) = E[(X - E[X])(Y - E[Y])] = E[XY] - E[X]E[Y]
```

## 三、统计推断的形式化系统

### 3.1 参数估计的形式化定义

**点估计的形式化定义**：

```
设 θ 是总体参数，θ̂ 是 θ 的估计量
θ̂: X → Θ，其中 X 是样本空间，Θ 是参数空间
```

**区间估计的形式化定义**：

```
置信区间 [L(X), U(X)] 满足：
P(L(X) ≤ θ ≤ U(X)) = 1 - α
其中 α 是显著性水平
```

### 3.2 假设检验的形式化定义

**假设检验的形式化定义**：

```
原假设 H₀: θ ∈ Θ₀
备择假设 H₁: θ ∈ Θ₁
其中 Θ₀ ∩ Θ₁ = ∅
```

**检验统计量的形式化定义**：

```
检验统计量 T(X) 是样本的函数
用于判断是否拒绝原假设
拒绝域 R = {x | T(x) > c}
其中 c 是临界值
```

### 3.3 回归分析的形式化定义

**线性回归模型的形式化定义**：

```
Y = β₀ + β₁X₁ + β₂X₂ + ... + βₚXₚ + ε
其中：
- Y 是因变量
- X₁, X₂, ..., Xₚ 是自变量
- β₀, β₁, ..., βₚ 是回归系数
- ε 是随机误差项
```

**最小二乘估计的形式化定义**：

```
β̂ = argmin Σᵢ(yᵢ - β₀ - β₁x₁ᵢ - ... - βₚxₚᵢ)²
```

## 四、概率统计推理系统

### 4.1 基本推理规则

**概率推理规则**：

```
1. 加法规则：P(A ∪ B) = P(A) + P(B) - P(A ∩ B)
2. 乘法规则：P(A ∩ B) = P(A)P(B|A)
3. 全概率公式：P(A) = ΣᵢP(A|Bᵢ)P(Bᵢ)
4. 贝叶斯公式：P(A|B) = P(B|A)P(A) / P(B)
```

**统计推理规则**：

```
1. 大数定律：样本均值收敛于总体均值
2. 中心极限定理：样本均值的分布近似正态分布
3. 似然原理：数据支持参数值的程度
```

### 4.2 统计检验方法

**参数检验**：

```
1. t检验：检验总体均值
2. F检验：检验方差比
3. χ²检验：检验分布拟合
4. z检验：大样本均值检验
```

**非参数检验**：

```
1. 符号检验：检验中位数
2. Wilcoxon秩和检验：检验分布差异
3. Kruskal-Wallis检验：多组比较
4. 卡方检验：分类数据检验
```
