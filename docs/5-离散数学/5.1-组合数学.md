# 离散数学概念形式模型

## 一、组合数学概念的形式化定义

### 1.1 排列的形式化定义

**排列的形式化定义**：

```
从 n 个不同元素中取 r 个元素的排列数
P(n, r) = n! / (n - r)!
其中 n! = n × (n-1) × ... × 2 × 1
性质：P(n, n) = n!，P(n, 0) = 1
```

**全排列的形式化定义**：

```
n 个不同元素的全排列数
P(n, n) = n!
符号表示：|Sₙ| = n!
其中 Sₙ 是 n 个元素的对称群
```

### 1.2 组合的形式化定义

**组合的形式化定义**：

```
从 n 个不同元素中取 r 个元素的组合数
C(n, r) = n! / (r! × (n - r)!)
符号表示：C(n, r) = (ⁿᵣ)
性质：C(n, r) = C(n, n-r)，C(n, 0) = C(n, n) = 1
```

**二项式系数的形式化定义**：

```
二项式系数 (ⁿᵣ) 满足：
(x + y)ⁿ = Σᵣ₌₀ⁿ (ⁿᵣ) xʳyⁿ⁻ʳ
性质：
1. (ⁿᵣ) = (ⁿⁿ⁻ʳ)
2. (ⁿᵣ) = (ⁿ⁻¹ᵣ) + (ⁿ⁻¹ᵣ⁻¹)
3. Σᵣ₌₀ⁿ (ⁿᵣ) = 2ⁿ
```

### 1.3 容斥原理的形式化定义

**容斥原理的形式化定义**：

```
设 A₁, A₂, ..., Aₙ 是有限集合
|∪ᵢ₌₁ⁿ Aᵢ| = Σᵢ|Aᵢ| - Σᵢ,ⱼ|Aᵢ ∩ Aⱼ| + Σᵢ,ⱼ,ₖ|Aᵢ ∩ Aⱼ ∩ Aₖ| - ... + (-1)ⁿ⁻¹|∩ᵢ₌₁ⁿ Aᵢ|
```

**概率容斥原理**：

```
P(∪ᵢ₌₁ⁿ Aᵢ) = ΣᵢP(Aᵢ) - Σᵢ,ⱼP(Aᵢ ∩ Aⱼ) + Σᵢ,ⱼ,ₖP(Aᵢ ∩ Aⱼ ∩ Aₖ) - ... + (-1)ⁿ⁻¹P(∩ᵢ₌₁ⁿ Aᵢ)
```

## 二、图论概念的形式化定义

### 2.1 图的基本概念

**图的形式化定义**：

```
图 G = (V, E) 由顶点集 V 和边集 E 组成
其中 E ⊆ V × V
符号表示：G = (V, E)
性质：图是顶点和边的集合
```

**有向图的形式化定义**：

```
有向图 G = (V, E) 由顶点集 V 和有序边集 E 组成
其中 E ⊆ V × V
符号表示：G = (V, E)
性质：边有方向性
```

**无向图的形式化定义**：

```
无向图 G = (V, E) 由顶点集 V 和边集 E 组成
其中 E ⊆ {{u, v} | u, v ∈ V, u ≠ v}
符号表示：G = (V, E)
性质：边无方向性
```

### 2.2 图的表示方法

**邻接矩阵的形式化定义**：

```
图 G = (V, E) 的邻接矩阵 A 是 n × n 矩阵
其中 n = |V|，A[i, j] = 1 如果 (i, j) ∈ E，否则 A[i, j] = 0
符号表示：A = [aᵢⱼ]ₙₓₙ
```

**邻接表的形式化定义**：

```
图 G = (V, E) 的邻接表是顶点到其邻居的映射
adj: V → 2^V
其中 adj(v) = {u ∈ V | (v, u) ∈ E}
```

### 2.3 图的性质

**连通性的形式化定义**：

```
图 G 是连通的，当且仅当对任意两个顶点 u, v ∈ V
存在从 u 到 v 的路径
符号表示：∀u, v ∈ V, ∃路径 P: u → v
```

**完全图的形式化定义**：

```
完全图 Kₙ 是 n 个顶点的图
其中任意两个不同顶点之间都有一条边
符号表示：Kₙ = (V, E)，其中 |V| = n，E = {{u, v} | u, v ∈ V, u ≠ v}
```

## 三、逻辑概念的形式化定义

### 3.1 命题逻辑

**命题的形式化定义**：

```
命题 p 是一个具有真值（真或假）的陈述句
符号表示：p ∈ {T, F}
其中 T 表示真，F 表示假
```

**逻辑连接词的形式化定义**：

```
否定：¬p（非p）
合取：p ∧ q（p且q）
析取：p ∨ q（p或q）
蕴含：p → q（p蕴含q）
等价：p ↔ q（p等价于q）
```

**真值表的形式化定义**：

```
命题公式的真值表是输入变量所有可能组合对应的输出值
对于 n 个变量，真值表有 2ⁿ 行
每行对应一种变量赋值组合
```

### 3.2 谓词逻辑

**谓词的形式化定义**：

```
谓词 P(x) 是包含变量的命题函数
其中 x 是变量，P 是谓词符号
符号表示：P: D → {T, F}
其中 D 是论域
```

**量词的形式化定义**：

```
全称量词：∀x P(x)（对所有x，P(x)成立）
存在量词：∃x P(x)（存在x，使得P(x)成立）
```

**谓词逻辑公式的形式化定义**：

```
原子公式：P(x₁, x₂, ..., xₙ)
复合公式：¬φ, φ ∧ ψ, φ ∨ ψ, φ → ψ, φ ↔ ψ
量化公式：∀x φ, ∃x φ
其中 φ, ψ 是谓词逻辑公式
```

### 3.3 证明方法

**直接证明的形式化定义**：

```
从前提 P 出发，通过逻辑推理直接得出结论 Q
符号表示：P ⊢ Q
步骤：
1. 假设 P 为真
2. 应用逻辑规则和已知定理
3. 得出结论 Q
```

**反证法的形式化定义**：

```
假设结论 Q 为假，通过推理得出矛盾
从而证明 Q 为真
符号表示：¬Q ⊢ ⊥，因此 Q
步骤：
1. 假设 ¬Q 为真
2. 从 ¬Q 出发进行推理
3. 得出矛盾 ⊥
4. 因此 Q 为真
```

**数学归纳法的形式化定义**：

```
证明对于所有自然数 n，命题 P(n) 成立
1. 基础步骤：证明 P(0) 成立
2. 归纳步骤：假设 P(k) 成立，证明 P(k+1) 成立
3. 结论：对所有 n ≥ 0，P(n) 成立
```

## 四、代数结构概念的形式化定义

### 4.1 群论

**群的形式化定义**：

```
群 (G, *) 是一个集合 G 和二元运算 *
满足以下公理：
1. 封闭性：∀a, b ∈ G, a * b ∈ G
2. 结合律：∀a, b, c ∈ G, (a * b) * c = a * (b * c)
3. 单位元：∃e ∈ G, ∀a ∈ G, e * a = a * e = a
4. 逆元：∀a ∈ G, ∃a⁻¹ ∈ G, a * a⁻¹ = a⁻¹ * a = e
```

**子群的形式化定义**：

```
群 (G, *) 的子群 (H, *) 满足：
1. H ⊆ G
2. ∀a, b ∈ H, a * b ∈ H
3. ∀a ∈ H, a⁻¹ ∈ H
4. e ∈ H
```

### 4.2 环论

**环的形式化定义**：

```
环 (R, +, *) 是一个集合 R 和两个二元运算 +, *
满足以下公理：
1. (R, +) 是阿贝尔群
2. (R, *) 是半群
3. 分配律：∀a, b, c ∈ R, a * (b + c) = a * b + a * c
```

**域的形式化定义**：

```
域 (F, +, *) 是一个环，其中：
1. (F, +) 是阿贝尔群
2. (F\{0}, *) 是阿贝尔群
3. 分配律成立
```

## 五、离散数学推理系统

### 5.1 组合推理规则

**乘法原理**：

```
如果事件 A 有 m 种方式，事件 B 有 n 种方式
且 A 和 B 独立，则 A 和 B 的组合有 m × n 种方式
符号表示：|A × B| = |A| × |B|
```

**加法原理**：

```
如果事件 A 有 m 种方式，事件 B 有 n 种方式
且 A 和 B 互斥，则 A 或 B 有 m + n 种方式
符号表示：|A ∪ B| = |A| + |B|（当 A ∩ B = ∅）
```

### 5.2 图论推理规则

**欧拉路径定理**：

```
无向图有欧拉路径当且仅当：
1. 图是连通的
2. 恰好有 0 或 2 个顶点的度数为奇数
```

**哈密顿路径定理**：

```
完全图 Kₙ 有哈密顿路径
符号表示：∀n ≥ 2, Kₙ 有哈密顿路径
```

### 5.3 逻辑推理规则

**逻辑等价规则**：

```
1. 双重否定：¬¬p ↔ p
2. 德摩根律：¬(p ∧ q) ↔ ¬p ∨ ¬q
3. 分配律：p ∧ (q ∨ r) ↔ (p ∧ q) ∨ (p ∧ r)
4. 蕴含等价：p → q ↔ ¬p ∨ q
```

**推理规则**：

```
1. 假言推理：p → q, p ⊢ q
2. 拒取式：p → q, ¬q ⊢ ¬p
3. 三段论：p → q, q → r ⊢ p → r
4. 析取三段论：p ∨ q, ¬p ⊢ q
```
