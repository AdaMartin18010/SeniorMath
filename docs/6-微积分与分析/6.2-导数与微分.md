# 导数与微分 | Derivatives and Differentials

## 概述

导数是微积分的核心概念，描述了函数在某一点的瞬时变化率。导数不仅在数学理论中有重要地位，在物理学、经济学、工程学等领域也有广泛应用。微分是导数的几何表示，为函数的局部线性近似提供了工具。

## 导数的定义

### 1. 导数的概念

```text
定义：函数f(x)在点x = a处的导数定义为：
f'(a) = lim(h→0) [f(a + h) - f(a)]/h
或
f'(a) = lim(x→a) [f(x) - f(a)]/(x - a)

几何意义：函数图像在点(a, f(a))处的切线斜率
物理意义：瞬时变化率，如瞬时速度、瞬时加速度
```

### 2. 导数的表示

```text
符号表示：
- f'(x)：拉格朗日记号
- df/dx：莱布尼茨记号
- Df(x)：算子记号
- ẏ：牛顿记号（对时间求导）

高阶导数：
- f''(x) = d²f/dx²：二阶导数
- f'''(x) = d³f/dx³：三阶导数
- f⁽ⁿ⁾(x) = dⁿf/dxⁿ：n阶导数
```

### 3. 导数的存在性

```text
必要条件：函数在该点连续
充分条件：函数在该点可导
反例：f(x) = |x|在x = 0处连续但不可导

可导的几何条件：函数图像在该点有唯一的切线
```

## 导数的计算

### 1. 基本导数公式

```text
常数函数：(c)' = 0
幂函数：(x^n)' = nx^(n-1)
指数函数：(e^x)' = e^x
对数函数：(ln x)' = 1/x
三角函数：
- (sin x)' = cos x
- (cos x)' = -sin x
- (tan x)' = sec² x
- (cot x)' = -csc² x
反三角函数：
- (arcsin x)' = 1/√(1 - x²)
- (arccos x)' = -1/√(1 - x²)
- (arctan x)' = 1/(1 + x²)
```

### 2. 导数的运算法则

```text
线性性质：
- (cf(x))' = cf'(x)
- (f(x) + g(x))' = f'(x) + g'(x)

乘法法则：(f(x)g(x))' = f'(x)g(x) + f(x)g'(x)
除法法则：(f(x)/g(x))' = [f'(x)g(x) - f(x)g'(x)]/g²(x)
链式法则：(f(g(x)))' = f'(g(x))g'(x)
反函数法则：(f⁻¹(x))' = 1/f'(f⁻¹(x))
```

### 3. 隐函数求导

```text
方法：对方程两边同时求导，然后解出dy/dx
例子：x² + y² = 1
两边求导：2x + 2y(dy/dx) = 0
解得：dy/dx = -x/y
```

### 4. 参数方程求导

```text
方法：dy/dx = (dy/dt)/(dx/dt)
例子：x = cos t, y = sin t
dy/dx = (cos t)/(-sin t) = -cot t
```

## 导数的应用

### 1. 几何应用

```text
切线方程：y - f(a) = f'(a)(x - a)
法线方程：y - f(a) = -1/f'(a)(x - a)
切点坐标：(a, f(a))
切线斜率：f'(a)
```

### 2. 物理应用

```text
瞬时速度：v(t) = ds/dt
瞬时加速度：a(t) = dv/dt = d²s/dt²
瞬时功率：P(t) = dW/dt
瞬时电流：i(t) = dq/dt
```

### 3. 经济学应用

```text
边际成本：MC = dC/dq
边际收益：MR = dR/dq
边际效用：MU = dU/dx
弹性：E = (dx/x)/(dp/p) = (p/x)(dx/dp)
```

## 函数的单调性

### 1. 单调性判定

```text
单调递增：f'(x) > 0
单调递减：f'(x) < 0
严格单调递增：f'(x) > 0
严格单调递减：f'(x) < 0
```

### 2. 极值判定

```text
必要条件：f'(a) = 0或f'(a)不存在
充分条件：
- 第一充分条件：f'(x)在a点两侧变号
- 第二充分条件：f'(a) = 0且f''(a) ≠ 0
```

### 3. 凹凸性判定

```text
凹函数：f''(x) > 0
凸函数：f''(x) < 0
拐点：f''(x) = 0且f''(x)在该点两侧变号
```

## 微分

### 1. 微分的定义

```text
定义：函数f(x)在点x处的微分定义为：
df = f'(x)dx

几何意义：函数在点x处的线性近似
物理意义：函数值的微小变化
```

### 2. 微分的性质

```text
线性性质：d(cf) = cdf, d(f + g) = df + dg
乘法法则：d(fg) = fdg + gdf
除法法则：d(f/g) = (gdf - fdg)/g²
链式法则：df(g(x)) = f'(g(x))dg(x)
```

### 3. 微分的应用

```text
近似计算：f(x + Δx) ≈ f(x) + f'(x)Δx
误差估计：|Δy - dy| ≤ M(Δx)²/2
线性化：用线性函数近似非线性函数
```

## 高阶导数

### 1. 高阶导数的概念

```text
二阶导数：f''(x) = d²f/dx²
三阶导数：f'''(x) = d³f/dx³
n阶导数：f⁽ⁿ⁾(x) = dⁿf/dxⁿ

几何意义：二阶导数表示曲线的凹凸性
物理意义：二阶导数表示加速度
```

### 2. 高阶导数的计算

```text
基本函数的高阶导数：
- (x^n)⁽ⁿ⁾ = n!
- (e^x)⁽ⁿ⁾ = e^x
- (sin x)⁽ⁿ⁾ = sin(x + nπ/2)
- (cos x)⁽ⁿ⁾ = cos(x + nπ/2)

莱布尼茨公式：(fg)⁽ⁿ⁾ = ΣC(n,k)f⁽ᵏ⁾g⁽ⁿ⁻ᵏ⁾
```

### 3. 高阶导数的应用

```text
泰勒展开：f(x) = Σf⁽ᵏ⁾(a)(x-a)ᵏ/k!
麦克劳林展开：f(x) = Σf⁽ᵏ⁾(0)xᵏ/k!
函数逼近：用多项式近似函数
```

## 导数的几何意义

### 1. 切线和法线

```text
切线：函数图像在点(a, f(a))处的切线方程为y - f(a) = f'(a)(x - a)
法线：函数图像在点(a, f(a))处的法线方程为y - f(a) = -1/f'(a)(x - a)
切点：切线与函数图像的交点
```

### 2. 函数的图像

```text
单调性：f'(x) > 0时函数递增，f'(x) < 0时函数递减
极值：f'(x) = 0的点可能是极值点
凹凸性：f''(x) > 0时函数凹，f''(x) < 0时函数凸
拐点：f''(x) = 0的点可能是拐点
```

### 3. 渐近线

```text
水平渐近线：lim(x→∞) f(x) = L，则y = L是水平渐近线
垂直渐近线：lim(x→a) f(x) = ∞，则x = a是垂直渐近线
斜渐近线：lim(x→∞) [f(x) - (kx + b)] = 0，则y = kx + b是斜渐近线
```

## 导数的应用问题

### 1. 最值问题

```text
步骤：
1. 求函数的导数
2. 令导数等于零，求驻点
3. 检查驻点和端点
4. 比较函数值，确定最值

应用：优化问题、经济学中的最优化
```

### 2. 相关变化率

```text
方法：
1. 建立变量之间的关系
2. 对关系式求导
3. 代入已知条件求解

例子：圆锥体积变化率、距离变化率
```

### 3. 微分方程

```text
可分离变量方程：dy/dx = f(x)g(y)
一阶线性方程：dy/dx + P(x)y = Q(x)
齐次方程：dy/dx = f(y/x)
```

## 常见错误和注意事项

### 1. 概念错误

```text
错误：认为导数就是函数值
正确：导数是函数值的变化率

错误：认为可导函数一定连续
正确：可导函数一定连续，但连续函数不一定可导
```

### 2. 计算错误

```text
错误：忘记链式法则
正确：复合函数求导必须使用链式法则

错误：混淆导数和微分
正确：导数是比值，微分是乘积
```

### 3. 理解误区

```text
误区：认为导数计算就是代数运算
事实：导数计算需要特殊的公式和法则

误区：认为所有函数都有导数
事实：有些函数在某些点不可导
```
