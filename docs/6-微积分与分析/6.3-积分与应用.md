# 积分与应用 | Integration and Applications

## 概述

积分是微积分的另一个核心概念，是导数的逆运算。积分不仅在数学理论中有重要地位，在物理学、工程学、经济学等领域也有广泛应用。积分提供了计算面积、体积、功、能量等物理量的强大工具。

## 不定积分

### 1. 不定积分的定义

```text
定义：如果F'(x) = f(x)，则称F(x)是f(x)的一个原函数，记作∫f(x)dx = F(x) + C
其中C是任意常数，称为积分常数

几何意义：原函数图像是f(x)图像下的面积函数
物理意义：累积量，如位移是速度的积分
```

### 2. 不定积分的性质

```text
线性性质：
- ∫[f(x) + g(x)]dx = ∫f(x)dx + ∫g(x)dx
- ∫cf(x)dx = c∫f(x)dx

积分与导数关系：
- d/dx[∫f(x)dx] = f(x)
- ∫f'(x)dx = f(x) + C
```

### 3. 基本积分公式

```text
幂函数：∫x^n dx = x^(n+1)/(n+1) + C（n ≠ -1）
指数函数：∫e^x dx = e^x + C
对数函数：∫1/x dx = ln|x| + C
三角函数：
- ∫sin x dx = -cos x + C
- ∫cos x dx = sin x + C
- ∫sec² x dx = tan x + C
- ∫csc² x dx = -cot x + C
反三角函数：
- ∫1/√(1-x²) dx = arcsin x + C
- ∫1/(1+x²) dx = arctan x + C
```

## 定积分

### 1. 定积分的定义

```text
定义：∫[a,b] f(x)dx = lim(n→∞) Σ[i=1,n] f(ξᵢ)Δxᵢ
其中[a,b]是积分区间，Δxᵢ = (b-a)/n，ξᵢ ∈ [xᵢ₋₁, xᵢ]

几何意义：函数f(x)在区间[a,b]上的曲线与x轴围成的面积
物理意义：累积量，如位移、功、能量等
```

### 2. 定积分的性质

```text
线性性质：
- ∫[a,b] [f(x) + g(x)]dx = ∫[a,b] f(x)dx + ∫[a,b] g(x)dx
- ∫[a,b] cf(x)dx = c∫[a,b] f(x)dx

区间性质：
- ∫[a,b] f(x)dx = -∫[b,a] f(x)dx
- ∫[a,b] f(x)dx = ∫[a,c] f(x)dx + ∫[c,b] f(x)dx

比较性质：
- 如果f(x) ≥ g(x)，则∫[a,b] f(x)dx ≥ ∫[a,b] g(x)dx
- |∫[a,b] f(x)dx| ≤ ∫[a,b] |f(x)|dx
```

### 3. 微积分基本定理

```text
第一基本定理：如果F(x) = ∫[a,x] f(t)dt，则F'(x) = f(x)
第二基本定理：∫[a,b] f(x)dx = F(b) - F(a)，其中F(x)是f(x)的原函数

意义：建立了积分与导数的关系，提供了计算定积分的有效方法
```

## 积分方法

### 1. 换元积分法

```text
第一类换元法：∫f(g(x))g'(x)dx = ∫f(u)du，其中u = g(x)
第二类换元法：∫f(x)dx = ∫f(g(t))g'(t)dt，其中x = g(t)

常用换元：
- 三角换元：x = a sin t, x = a tan t, x = a sec t
- 指数换元：x = e^t
- 对数换元：x = ln t
```

### 2. 分部积分法

```text
公式：∫u dv = uv - ∫v du
选择原则：LIATE法则
- L：对数函数
- I：反三角函数
- A：代数函数
- T：三角函数
- E：指数函数

应用：∫x e^x dx, ∫x ln x dx, ∫x sin x dx
```

### 3. 有理函数积分

```text
方法：部分分式分解
步骤：
1. 将有理函数分解为部分分式
2. 对每个部分分式积分
3. 合并结果

例子：∫(x+1)/(x²-1) dx
分解：(x+1)/(x²-1) = 1/(x-1)
积分：∫1/(x-1) dx = ln|x-1| + C
```

### 4. 三角积分

```text
基本三角积分：
- ∫sin x dx = -cos x + C
- ∫cos x dx = sin x + C
- ∫tan x dx = -ln|cos x| + C
- ∫cot x dx = ln|sin x| + C

三角恒等式应用：
- sin² x = (1 - cos 2x)/2
- cos² x = (1 + cos 2x)/2
- sin x cos x = sin 2x/2
```

## 积分的应用

### 1. 几何应用

```text
平面图形面积：
- 曲线y = f(x)与x轴围成的面积：A = ∫[a,b] |f(x)|dx
- 两条曲线围成的面积：A = ∫[a,b] |f(x) - g(x)|dx

旋转体体积：
- 绕x轴旋转：V = π∫[a,b] f²(x)dx
- 绕y轴旋转：V = 2π∫[a,b] x|f(x)|dx

弧长：L = ∫[a,b] √(1 + [f'(x)]²)dx
```

### 2. 物理应用

```text
位移：s = ∫v(t)dt
功：W = ∫F(x)dx
能量：E = ∫P(t)dt
电荷：Q = ∫i(t)dt
质量：m = ∫ρ(x)dx
```

### 3. 经济学应用

```text
总成本：TC = ∫MC(q)dq
总收益：TR = ∫MR(q)dq
消费者剩余：CS = ∫[0,Q] [P(q) - P*]dq
生产者剩余：PS = ∫[0,Q] [P* - MC(q)]dq
```

## 数值积分

### 1. 矩形法

```text
公式：∫[a,b] f(x)dx ≈ Σ[i=1,n] f(xᵢ)Δx
其中Δx = (b-a)/n，xᵢ = a + iΔx

精度：误差约为O(1/n)
```

### 2. 梯形法

```text
公式：∫[a,b] f(x)dx ≈ (Δx/2)[f(x₀) + 2f(x₁) + ... + 2f(xₙ₋₁) + f(xₙ)]
其中Δx = (b-a)/n，xᵢ = a + iΔx

精度：误差约为O(1/n²)
```

### 3. 辛普森法

```text
公式：∫[a,b] f(x)dx ≈ (Δx/3)[f(x₀) + 4f(x₁) + 2f(x₂) + 4f(x₃) + ... + 2f(xₙ₋₂) + 4f(xₙ₋₁) + f(xₙ)]

精度：误差约为O(1/n⁴)
```

## 广义积分

### 1. 无穷积分

```text
定义：∫[a,∞) f(x)dx = lim(b→∞) ∫[a,b] f(x)dx
收敛条件：极限存在且有限

例子：
- ∫[1,∞) 1/x² dx = 1（收敛）
- ∫[1,∞) 1/x dx = ∞（发散）
```

### 2. 瑕积分

```text
定义：∫[a,b] f(x)dx = lim(ε→0⁺) ∫[a+ε,b] f(x)dx
其中f(x)在x = a处无界

收敛条件：极限存在且有限

例子：
- ∫[0,1] 1/√x dx = 2（收敛）
- ∫[0,1] 1/x dx = ∞（发散）
```

### 3. 收敛性判别

```text
比较判别法：如果0 ≤ f(x) ≤ g(x)，且∫g(x)dx收敛，则∫f(x)dx收敛
比值判别法：如果lim(x→∞) f(x+1)/f(x) < 1，则∫f(x)dx收敛
积分判别法：对于单调递减的正函数，∫f(x)dx与Σf(n)同敛散
```

## 多重积分

### 1. 二重积分

```text
定义：∬[D] f(x,y)dxdy = lim(λ→0) Σ[i=1,n] f(ξᵢ,ηᵢ)Δσᵢ
其中D是积分区域，Δσᵢ是面积元素

几何意义：函数f(x,y)在区域D上的体积
```

### 2. 三重积分

```text
定义：∭[V] f(x,y,z)dxdydz
其中V是积分区域

物理意义：密度函数在体积V上的质量
```

### 3. 积分区域

```text
直角坐标：dxdy, dxdydz
极坐标：rdrdθ
柱坐标：rdrdθdz
球坐标：r²sinφdrdθdφ
```

## 常见错误和注意事项

### 1. 概念错误

```text
错误：认为积分就是面积
正确：积分是累积量，面积只是几何意义

错误：忘记积分常数
正确：不定积分必须加上积分常数
```

### 2. 计算错误

```text
错误：积分时忘记换元
正确：复杂积分需要适当的换元

错误：定积分忘记上下限
正确：定积分必须明确积分区间
```

### 3. 理解误区

```text
误区：认为所有函数都可积
事实：有些函数不可积

误区：认为积分计算就是代数运算
事实：积分计算需要特殊的技巧和方法
```
