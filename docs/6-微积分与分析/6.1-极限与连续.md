## 极限与连续 | Limits and Continuity

## 概述

极限是微积分的核心概念，是连接代数与分析的重要桥梁。极限理论不仅为导数和积分奠定基础，还为理解函数的性质提供了重要工具。连续性是函数的重要性质，在实际应用中具有重要意义。

## 数列极限

### 1. 数列极限的定义

```text
定义：对于数列{aₙ}，如果存在实数L，使得对于任意ε > 0，存在正整数N，当n > N时，有|aₙ - L| < ε，则称数列{aₙ}收敛于L，记作lim(n→∞) aₙ = L
几何意义：当n充分大时，数列项与L的距离可以任意小
例子：
- lim(n→∞) 1/n = 0
- lim(n→∞) (1 + 1/n)^n = e
- lim(n→∞) n/(n+1) = 1
```

### 2. 数列极限的性质

```text
唯一性：如果数列收敛，则极限唯一
有界性：收敛数列必有界
保号性：如果lim aₙ = L > 0，则存在N，当n > N时，aₙ > 0
夹逼定理：如果aₙ ≤ bₙ ≤ cₙ，且lim aₙ = lim cₙ = L，则lim bₙ = L
```

### 3. 数列极限的运算

```text
四则运算：
- lim(aₙ + bₙ) = lim aₙ + lim bₙ
- lim(aₙ - bₙ) = lim aₙ - lim bₙ
- lim(aₙ × bₙ) = lim aₙ × lim bₙ
- lim(aₙ / bₙ) = lim aₙ / lim bₙ（lim bₙ ≠ 0）

复合运算：
- lim f(aₙ) = f(lim aₙ)（f连续）
```

## 函数极限

### 1. 函数极限的定义

```text
定义：对于函数f(x)，如果存在实数L，使得对于任意ε > 0，存在δ > 0，当0 < |x - a| < δ时，有|f(x) - L| < ε，则称函数f(x)在x = a处的极限为L，记作lim(x→a) f(x) = L
几何意义：当x充分接近a时，f(x)与L的距离可以任意小
```

### 2. 单侧极限

```text
左极限：lim(x→a⁻) f(x) = L，表示x从左侧趋近于a时f(x)的极限
右极限：lim(x→a⁺) f(x) = L，表示x从右侧趋近于a时f(x)的极限
关系：lim(x→a) f(x) = L 当且仅当 lim(x→a⁻) f(x) = lim(x→a⁺) f(x) = L
```

### 3. 无穷极限

```text
x → ∞：lim(x→∞) f(x) = L
x → -∞：lim(x→-∞) f(x) = L
f(x) → ∞：lim(x→a) f(x) = ∞
f(x) → -∞：lim(x→a) f(x) = -∞
```

### 4. 函数极限的性质

```text
唯一性：如果极限存在，则极限唯一
局部有界性：如果lim(x→a) f(x) = L，则f(x)在a的某个邻域内有界
保号性：如果lim(x→a) f(x) = L > 0，则存在δ > 0，当0 < |x - a| < δ时，f(x) > 0
夹逼定理：如果f(x) ≤ g(x) ≤ h(x)，且lim(x→a) f(x) = lim(x→a) h(x) = L，则lim(x→a) g(x) = L
```

## 极限的计算

### 1. 基本极限

```text
常数极限：lim(x→a) c = c
恒等函数极限：lim(x→a) x = a
幂函数极限：lim(x→a) x^n = a^n
指数函数极限：lim(x→a) e^x = e^a
对数函数极限：lim(x→a) ln x = ln a（a > 0）
三角函数极限：lim(x→0) sin x/x = 1
```

### 2. 极限的运算

```text
四则运算：
- lim(x→a) [f(x) + g(x)] = lim(x→a) f(x) + lim(x→a) g(x)
- lim(x→a) [f(x) - g(x)] = lim(x→a) f(x) - lim(x→a) g(x)
- lim(x→a) [f(x) × g(x)] = lim(x→a) f(x) × lim(x→a) g(x)
- lim(x→a) [f(x) / g(x)] = lim(x→a) f(x) / lim(x→a) g(x)（lim(x→a) g(x) ≠ 0）

复合运算：
- lim(x→a) f(g(x)) = f(lim(x→a) g(x))（f连续）
```

### 3. 不定式

```text
0/0型：使用因式分解、洛必达法则等
∞/∞型：使用洛必达法则、比较阶数等
0×∞型：转化为0/0或∞/∞型
∞-∞型：通分或提取公因式
0^0型：使用对数或指数变换
∞^0型：使用对数或指数变换
1^∞型：使用对数或指数变换
```

### 4. 重要极限

```text
lim(x→0) sin x/x = 1
lim(x→0) (1 - cos x)/x² = 1/2
lim(x→∞) (1 + 1/x)^x = e
lim(x→0) (e^x - 1)/x = 1
lim(x→0) ln(1 + x)/x = 1
```

## 连续性

### 1. 连续性的定义

```text
定义：函数f(x)在点x = a处连续，如果满足：
1. f(a)存在
2. lim(x→a) f(x)存在
3. lim(x→a) f(x) = f(a)

等价定义：对于任意ε > 0，存在δ > 0，当|x - a| < δ时，有|f(x) - f(a)| < ε
几何意义：函数图像在点(a, f(a))处没有"断点"
```

### 2. 连续性的类型

```text
左连续：lim(x→a⁻) f(x) = f(a)
右连续：lim(x→a⁺) f(x) = f(a)
区间连续：函数在区间内每一点都连续
一致连续：对于任意ε > 0，存在δ > 0，当|x₁ - x₂| < δ时，有|f(x₁) - f(x₂)| < ε
```

### 3. 连续函数的性质

```text
局部有界性：连续函数在任意点的邻域内有界
保号性：如果f(a) > 0，则存在δ > 0，当|x - a| < δ时，f(x) > 0
介值定理：如果f(x)在[a, b]上连续，且f(a) ≠ f(b)，则对于f(a)与f(b)之间的任意值c，存在ξ ∈ (a, b)，使得f(ξ) = c
最值定理：如果f(x)在[a, b]上连续，则f(x)在[a, b]上有最大值和最小值
```

### 4. 初等函数的连续性

```text
基本初等函数：
- 幂函数：x^n在定义域上连续
- 指数函数：e^x在R上连续
- 对数函数：ln x在(0, +∞)上连续
- 三角函数：sin x, cos x在R上连续
- 反三角函数：arcsin x, arccos x在定义域上连续

复合函数：如果f(x)在x = a处连续，g(x)在y = f(a)处连续，则g(f(x))在x = a处连续
```

## 间断点

### 1. 间断点的分类

```text
第一类间断点：
- 可去间断点：lim(x→a⁻) f(x) = lim(x→a⁺) f(x) ≠ f(a)
- 跳跃间断点：lim(x→a⁻) f(x) ≠ lim(x→a⁺) f(x)

第二类间断点：
- 无穷间断点：lim(x→a) f(x) = ∞
- 振荡间断点：lim(x→a) f(x)不存在且不为∞
```

### 2. 间断点的判定

```text
判定步骤：
1. 检查f(a)是否存在
2. 检查lim(x→a⁻) f(x)和lim(x→a⁺) f(x)是否存在
3. 比较极限值与函数值
4. 根据比较结果确定间断点类型
```

## 极限的应用

### 1. 函数性质研究

```text
渐近线：
- 水平渐近线：y = L，其中lim(x→∞) f(x) = L
- 垂直渐近线：x = a，其中lim(x→a) f(x) = ∞
- 斜渐近线：y = kx + b，其中k = lim(x→∞) f(x)/x，b = lim(x→∞) [f(x) - kx]

函数图像：利用极限研究函数的整体性质
```

### 2. 实际问题

```text
瞬时速度：v(t) = lim(Δt→0) [s(t + Δt) - s(t)]/Δt
瞬时变化率：f'(x) = lim(h→0) [f(x + h) - f(x)]/h
面积计算：利用极限定义定积分
```

### 3. 数值计算

```text
近似计算：利用极限进行数值近似
收敛性判断：判断级数或积分的收敛性
误差估计：估计近似计算的误差
```

## 常见错误和注意事项

### 1. 概念错误

```text
错误：认为极限就是函数值
正确：极限是函数值的一种趋势

错误：认为连续函数一定有导数
正确：连续函数不一定有导数（如|x|在x = 0处）
```

### 2. 计算错误

```text
错误：直接代入导致0/0
正确：使用因式分解、洛必达法则等方法

错误：忽略极限的存在性
正确：先判断极限是否存在再计算
```

### 3. 理解误区

```text
误区：认为极限计算就是代数运算
事实：极限计算需要特殊的技巧和方法

误区：认为连续函数图像一定是光滑的
事实：连续函数图像可能有"尖点"
```
