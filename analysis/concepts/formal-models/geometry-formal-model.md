# 几何概念形式模型

## 一、基本几何概念形式化定义

### 1.1 点、线、面的形式化定义

**点的形式化定义**：

```text
点 P ∈ ℝ² 或 P ∈ ℝ³
表示为：P = (x, y) 或 P = (x, y, z)
其中 x, y, z ∈ ℝ
```

**直线的形式化定义**：

```text
直线 l 在平面上的形式化定义：
l = {(x, y) | ax + by + c = 0, a, b, c ∈ ℝ, a² + b² ≠ 0}

直线 l 在空间中的形式化定义：
l = {(x, y, z) | x = x₀ + at, y = y₀ + bt, z = z₀ + ct, t ∈ ℝ}
其中 (x₀, y₀, z₀) 是直线上一点，向量 (a, b, c) 是方向向量
```

**平面的形式化定义**：

```text
平面 π 在空间中的形式化定义：
π = {(x, y, z) | ax + by + cz + d = 0, a, b, c, d ∈ ℝ, a² + b² + c² ≠ 0}
其中 (a, b, c) 是平面的法向量
```

### 1.2 角的形式化定义

**角的形式化定义**：

```text
角 ∠ABC 的形式化定义：
∠ABC = {θ | θ = arccos(⟨AB, BC⟩ / (|AB| · |BC|)), 0 ≤ θ ≤ π}
其中 AB, BC 是向量，⟨AB, BC⟩ 是内积
```

**角的分类**：

```text
锐角：0 < θ < π/2
直角：θ = π/2
钝角：π/2 < θ < π
平角：θ = π
```

### 1.3 多边形和圆的形式化定义

**多边形的形式化定义**：

```text
n边形 P 的形式化定义：
P = {P₁, P₂, ..., Pₙ} 其中 Pᵢ ∈ ℝ²
满足：
1. Pᵢ ≠ Pⱼ 当 i ≠ j
2. 线段 PᵢPᵢ₊₁ 不相交（除端点外）
3. Pₙ₊₁ = P₁
```

**圆的形式化定义**：

```text
圆 C 的形式化定义：
C = {(x, y) | (x - a)² + (y - b)² = r²}
其中 (a, b) 是圆心，r > 0 是半径
```

## 二、几何变换形式化定义

### 2.1 平移变换

**平移变换的形式化定义**：

```text
平移变换 Tᵥ: ℝ² → ℝ²
Tᵥ(P) = P + v
其中 v = (a, b) 是平移向量
```

**平移变换的性质**：

```text
1. 保持距离：d(Tᵥ(P), Tᵥ(Q)) = d(P, Q)
2. 保持角度：∠Tᵥ(A)Tᵥ(B)Tᵥ(C) = ∠ABC
3. 保持面积：Area(Tᵥ(S)) = Area(S)
```

### 2.2 旋转变换

**旋转变换的形式化定义**：

```text
旋转变换 R_θ: ℝ² → ℝ²
R_θ(x, y) = (x cos θ - y sin θ, x sin θ + y cos θ)
其中 θ 是旋转角度
```

**旋转变换的性质**：

```text
1. 保持距离：d(R_θ(P), R_θ(Q)) = d(P, Q)
2. 保持角度：∠R_θ(A)R_θ(B)R_θ(C) = ∠ABC
3. 保持面积：Area(R_θ(S)) = Area(S)
4. 保持方向：det(R_θ) = 1
```

### 2.3 缩放变换

**缩放变换的形式化定义**：

```text
缩放变换 S_k: ℝ² → ℝ²
S_k(x, y) = (kx, ky)
其中 k > 0 是缩放因子
```

**缩放变换的性质**：

```text
1. 距离缩放：d(S_k(P), S_k(Q)) = k · d(P, Q)
2. 面积缩放：Area(S_k(S)) = k² · Area(S)
3. 角度保持不变：∠S_k(A)S_k(B)S_k(C) = ∠ABC
```

## 三、几何定理形式化证明

### 3.1 勾股定理的形式化证明

**勾股定理**：

```text
在直角三角形中：a² + b² = c²
其中 a, b 是直角边，c 是斜边
```

**形式化证明**：

```text
设直角三角形顶点为 A, B, C，直角在 C
则：
AB² = AC² + BC²

证明：
设 A = (0, 0), B = (a, 0), C = (0, b)
则：
AB² = a² + b²
AC² = b²
BC² = a²
因此：AB² = AC² + BC²
```

### 3.2 相似三角形的形式化证明

**相似三角形的形式化定义**：

```text
三角形 ΔABC ∽ ΔDEF 当且仅当：
1. ∠A = ∠D, ∠B = ∠E, ∠C = ∠F
2. AB/DE = BC/EF = AC/DF = k
其中 k 是相似比
```

**相似三角形的性质**：

```text
1. 对应角相等
2. 对应边成比例
3. 面积比等于相似比的平方
4. 周长比等于相似比
```

## 四、立体几何形式化定义

### 4.1 空间几何体的形式化定义

**立方体的形式化定义**：

```text
立方体 C = {(x, y, z) | 0 ≤ x, y, z ≤ a}
其中 a > 0 是边长
```

**球体的形式化定义**：

```text
球体 S = {(x, y, z) | (x - a)² + (y - b)² + (z - c)² ≤ r²}
其中 (a, b, c) 是球心，r > 0 是半径
```

**圆柱体的形式化定义**：

```text
圆柱体 Cyl = {(x, y, z) | (x - a)² + (y - b)² ≤ r², 0 ≤ z ≤ h}
其中 (a, b) 是底面圆心，r > 0 是底面半径，h > 0 是高度
```

### 4.2 表面积和体积的形式化定义

**立方体的表面积和体积**：

```text
表面积：S = 6a²
体积：V = a³
```

**球体的表面积和体积**：

```text
表面积：S = 4πr²
体积：V = (4/3)πr³
```

**圆柱体的表面积和体积**：

```text
表面积：S = 2πr² + 2πrh
体积：V = πr²h
```

## 五、解析几何形式化定义

### 5.1 坐标系的形式化定义

**直角坐标系**：

```text
平面直角坐标系：{(x, y) | x, y ∈ ℝ}
空间直角坐标系：{(x, y, z) | x, y, z ∈ ℝ}
```

**极坐标系**：

```text
平面极坐标系：{(r, θ) | r ≥ 0, 0 ≤ θ < 2π}
其中 r 是极径，θ 是极角
```

### 5.2 曲线的形式化定义

**直线的解析几何定义**：

```text
一般式：ax + by + c = 0
斜截式：y = kx + b
点斜式：y - y₁ = k(x - x₁)
```

**圆的解析几何定义**：

```text
标准式：(x - a)² + (y - b)² = r²
一般式：x² + y² + Dx + Ey + F = 0
其中 D² + E² - 4F > 0
```

**椭圆的解析几何定义**：

```text
标准式：(x/a)² + (y/b)² = 1
其中 a > b > 0
```

## 六、向量几何形式化定义

### 6.1 向量的形式化定义

**向量的形式化定义**：

```text
向量 v ∈ ℝⁿ 表示为：v = (v₁, v₂, ..., vₙ)
其中 vᵢ ∈ ℝ, i = 1, 2, ..., n
```

**向量的运算**：

```text
加法：u + v = (u₁ + v₁, u₂ + v₂, ..., uₙ + vₙ)
数乘：k · v = (kv₁, kv₂, ..., kvₙ)
内积：⟨u, v⟩ = u₁v₁ + u₂v₂ + ... + uₙvₙ
```

### 6.2 向量几何的应用

**向量表示直线**：

```text
直线 l 的向量方程：
r = r₀ + tv
其中 r₀ 是直线上一点，v 是方向向量，t ∈ ℝ
```

**向量表示平面**：

```text
平面 π 的向量方程：
⟨r - r₀, n⟩ = 0
其中 r₀ 是平面上一点，n 是法向量
```

## 七、几何推理系统

### 7.1 基本推理规则

**等量代换规则**：

```text
如果 a = b 且 b = c，则 a = c
```

**传递性规则**：

```text
如果 a > b 且 b > c，则 a > c
```

**对称性规则**：

```text
如果 a = b，则 b = a
```

### 7.2 几何证明方法

**直接证明法**：

```text
从已知条件出发，通过逻辑推理直接得出结论
```

**反证法**：

```text
假设结论不成立，推导出矛盾，从而证明结论成立
```

**归纳法**：

```text
证明 n = 1 时成立，假设 n = k 时成立，证明 n = k + 1 时也成立
```

### 7.3 几何推理的应用

**距离计算**：

```text
两点间距离：d(P, Q) = √[(x₂ - x₁)² + (y₂ - y₁)²]
```

**角度计算**：

```text
两向量夹角：cos θ = ⟨u, v⟩ / (|u| · |v|)
```

**面积计算**：

```text
三角形面积：S = (1/2) |det([AB, AC])|
其中 AB, AC 是向量
```
