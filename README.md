# SeniorMath - 高中数学知识体系梳理项目

## 项目定位

**SeniorMath** 是一个专注于高中数学知识体系梳理的教育项目。本项目致力于系统化、结构化、标准化地梳理高中数学知识，建立完整的知识关联网络，提供高质量的教育资源，对齐国际数学教育标准。

### 核心价值

- **知识梳理**：系统梳理高中数学知识体系
- **关联构建**：建立完整的知识关联网络
- **资源建设**：提供高质量的教育资源
- **标准对齐**：对齐国际数学教育标准

---

- **项目类型**：知识梳理与教育资源建设

## 知识关联网络

### 核心概念网络

- **函数概念网络**：贯穿代数、几何、微积分
- **极限概念网络**：连接微积分、数列、函数
- **向量概念网络**：连接几何、代数、物理应用
- **概率概念网络**：连接统计、数据分析、决策理论

### 跨模块桥梁

- **代数与几何**：解析几何、向量代数
- **代数与微积分**：函数分析、极限理论
- **几何与微积分**：曲线积分、面积计算
- **概率与统计**：数据分析、统计推断
